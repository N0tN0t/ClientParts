Практическая работа №6: Анимация в CSS

Демонстрация трансформации розового круга в синий квадрат с промежуточным фиолетовым прямоугольником.

Код анимации:

@keyframes circle-to-square {
    from {
        width: 100px;
        height: 100px;
        background-color: #f38ba8;
        border-radius: 50%;
    }
    50% {
        width: 150px;
        height: 80px;
        background-color: #fab387;
        border-radius: 20%;
    }
    to {
        width: 100px;
        height: 100px;
        background-color: #89b4fa;
        border-radius: 0;
    }
}

.anim-circle {
    width: 100px;
    height: 100px;
    background-color: #f38ba8;
    border-radius: 50%;
    animation: circle-to-square 3s infinite alternate ease-in-out 0.5s both;
}

Свойства анимации:

  • animation-name: circle-to-square
  • animation-duration: 3s
  • animation-iteration-count: infinite
  • animation-direction: alternate
  • animation-timing-function: ease-in-out
  • animation-delay: 0.5s
  • animation-fill-mode: both

При наведении — анимация останавливается, элемент поворачивается и увеличивается.

Практическая работа №5: Flexbox + Медиа-запросы

Адаптивная сетка блоков с использованием Flexible Box Layout.

Блокчейн

Распределённый реестр транзакций, защищённый криптографией.

Хеш

Уникальный цифровой отпечаток данных фиксированной длины.

Майнинг

Процесс подтверждения транзакций и создания новых блоков.

Медиа-запросы в действии:

  • ≤768px: Вертикальная компоновка
  • ≥1024px: Равномерные колонки
  • print: Убираются анимации и тени
  • landscape: Зелёная подсветка заметок
  • portrait: Розовая подсветка заметок

Текущая ориентация: Поверните устройство — цвет заметки изменится!
